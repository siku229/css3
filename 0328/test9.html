Display 속성

Display 속성은 각각의 요소를 어떻게 보여줄지 결정한다.

속성처리값? visibility (보여주기 속성 - 기본값) / hidden (감추기) / inherit (부모요소의 값을 상속)

none : 보여주지 않을때
inline : 인라인상태(자신의 고유영역만 확보) - 단어의 의미로 해석
block : 박스영역을 차지하는 속성 - 문단의 의미로 해석
inline-block : inline과 block의 합성
